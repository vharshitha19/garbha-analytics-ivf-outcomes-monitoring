# Garbha Analytics: Data-Driven IVF Outcomes Monitoring

## Project Overview

Garbha Analytics is a data analytics project focused on analyzing IVF (In Vitro Fertilization) treatment outcomes using Python and Power BI. The project aims to identify the key factors influencing IVF success rates and present meaningful insights through interactive dashboards.

---

## Objectives

- Analyze IVF treatment records
- Identify factors affecting treatment success
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Build interactive Power BI dashboards
- Generate actionable insights for data-driven decision making

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI
- Microsoft Excel

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Data Preprocessing
4. Exploratory Data Analysis
5. Feature Analysis
6. Dashboard Development
7. Insight Generation

---

## Key Analyses Performed

- Age-wise IVF success rate analysis
- Treatment cycle analysis
- Patient demographic analysis
- Clinical factor analysis
- Outcome comparison
- Trend analysis

---

## Dashboard Features

- Interactive filters
- KPI cards
- Success rate visualization
- Treatment trends
- Patient demographic insights

---

## Results

The project identified important trends and relationships among various clinical and demographic factors influencing IVF treatment outcomes. Interactive Power BI dashboards were developed to help visualize these insights effectively.
